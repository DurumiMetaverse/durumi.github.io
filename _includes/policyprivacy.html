1. 그룹관리 1.1 서버: 데이터베이스 구성, 그룹아이디(gid) 발급, 노인식별번호(uid)부여, 그룹삭제 1.2 복지사 애플리케이션: 메인 화면 구성, 그룹 값 입력, OTP 출력, 그룹참여 인원 확인 1.3 노인 애플리케이션: 메인화면 구성, 그룹참여 2. 위치 2.1 서버: 위치정보 관리 2.2 복지사 애플리케이션: 지도출력, 지도 안의 복지사와 노인 식별, 인솔범위 표시 2.3 노인 애플리케이션: 노인 위치 전송, 복지사 위치정보 수집 3. 이탈 3.1 서버: 이탈 감지, 이탈 상태 업데이트 3.2 복지사 애플리케이션: 이탈자 정보, 이탈자 변화 표시 3.3 노인 애플리케이션: 이탈 발생 알림, 이탈 발생 경고 아이콘, 복귀 방향 표시 4. 집결 4.1 서버: 집결 장소 전파 4.2 복지사 애플리케이션: 그룹연장, 집결 설정, 집결 인원관리 4.3 노인 애플리케이션: 복귀 방향 정보 변경, 도움 정보 출력, 집결 요청 출력
